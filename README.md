# OpenGram

An Android application project.

## Project Setup

This is an Android app repository built with Gradle.

### Requirements

- Android SDK
- Java Development Kit (JDK)
- Gradle

### Building

```bash
./gradlew build
```

### Running Tests

```bash
./gradlew test
```

## Structure

- `app/` - Main application module
- `gradle/` - Gradle wrapper and build scripts
- `local.properties` - Local SDK/NDK paths (not committed)
